# StyleForMen - 남성 패션 스타일 추천 서비스

사진을 업로드하고 키/몸무게를 입력하면, 체형을 분석하여 어울리는 패션 스타일을 추천해드리는 웹 서비스입니다.

## 주요 기능
- 📷 사진 업로드 (드래그 앤 드롭 지원)
- 📐 BMI 기반 체형 분석 (5가지 체형 분류)
- 👔 맞춤 스타일 추천 (캐주얼/비즈니스/스트릿)
- 🎨 체형별 컬러 팔레트 추천
- 💡 실용적인 스타일링 팁 제공

## 사용법
1. `index.html` 파일을 브라우저에서 열기
2. 사진 업로드 → 키/몸무게 입력 → 추천 결과 확인

## 기술 스택
- HTML5 / CSS3 / Vanilla JavaScript
- 서버 없이 브라우저에서 완전히 동작
